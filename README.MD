Este proyecto está hecho con:

Vue 3 (Composition API) - Framework JavaScript
TypeScript - Lenguaje de programación
Pinia - Gestor de estado (reemplazo de Vuex)
Vue Router - Enrutamiento
Axios - Cliente HTTP para consumir la PokeAPI
CSS - Estilos personalizados
Es una aplicación Pokédex que consume datos de la PokeAPI para mostrar información de los 151 Pokémon originales.
